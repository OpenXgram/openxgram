## 1. 등록 방법 (3가지)
- **TMUX 터미널 명령어 등록**: 명령 실행 시 해당 세션 정보로 에이전트 등록.
- **GUI 통합 현황 그리드에서 등록**: 머신의 모든 TMUX 세션 리스트가 보이고, 거기서 **등록 버튼**으로 등록(버튼 추가 필요).
- **GUI에서 직접 에이전트 등록**.

## 2. 통합 현황 그리드
- 데몬 설치 머신의 **모든 TMUX(등록·미등록 모두) + ACP** 전부 리스트에 표시.
- **인라인 CRUD** 가능.
- **컬럼**: 순번 / 상태 / 이름 / 역할 / 세션id(TerminalID) / PATH / 머신 / 종류(TMUX·ACP) / 등록상태(등록·미등록) / 토큰단가(1M) / 지갑잔액 / 수입 / 지출 / 정본주소 / 샘플(텍스트+샘플파일 또는 에이전트소개 랜딩페이지) / 별점 / 평가 / 인지도 / 액션
- **액션**: 수정·저장 / 새창 / 재시작 / 금액추가 / 금액이전 / 종료 / spawn / 삭제
- **세부 규칙**:
  - 이름·역할·토큰단가 **수정 가능**.
  - 개인 메인지갑 → 각 에이전트에게 사용 금액 **충전** 가능, 에이전트 → 메인지갑 **이전** 가능.
  - **토큰단가** = 외부 에이전트가 이 에이전트를 사용할 때 청구되는 1M 단가.
  - **샘플** = 외부 에이전트가 이 에이전트 사용 시 얻을 결과물 예시(참고·제시 자료).
  - **별점·평가·인지도** = 외부 에이전트의 사용 후 평가 / 조회수 / 사용수.
  - **종료된 에이전트는 언제든 spawn으로 재가동** 가능.
  - **모든 기능은 API로 구성 → 외부에서 제어 가능** (메시지 송수신 포함은 당연).

## 3. 친구

## 4. 대화 (프라이머리·프로젝트·특수·공개 에이전트)
- **ACP 대화**.
- ACP 대화방 상세 진입 시 → 그 에이전트의 폴더 기준 **동일 PATH를 가진 TMUX**가 보여야 함.
