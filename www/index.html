Core values

어떤 AI 서비스든, 같은 기억 — 외부에 정보 노출 없이

Any AI service, same memory — with zero data exposure

Bring Your Own LLM

모델을 바꿔도 기억과 자격이 따라옵니다

Switch models, keep memory and credentials

Claude → GPT → Gemini → 로컬 모델. 같은 에이전트가 모델만 바꿔도 같은 기억과 같은 자격증명을 그대로 사용합니다. 락인 제로.

Claude → GPT → Gemini → local. Swap the model, the agent keeps the same memory and the same credentials. Zero lock-in.

Sovereign data

모든 데이터가 사용자 머신에 남습니다

All data stays on your machine

로컬 SQLite + sqlite-vec. 외부 클라우드에 메모리·키를 보내지 않습니다. Nostr 동기화도 자체 호스팅 relay 가능. 데이터 주권은 사용자 손 안에.

Local SQLite + sqlite-vec. No memory or keys ever leave for a third-party cloud. Even Nostr sync runs on your own relay. Sovereignty stays with you.

Programmable trust

에이전트가 넘지 못할 경계

Boundaries the agent cannot cross

vault ACL · 일일 한도 · 승인 정책(auto / confirm / mfa) · 블록체인 암호화 기록. 자율 에이전트에게 권한을 주되, 결코 통제를 잃지 않습니다.

Vault ACL · daily limits · approval policy (auto / confirm / mfa) · blockchain-encrypted log. Give autonomous agents power without losing control.

Standards-aligned

표준 준수 — 자물쇠 없는 인프라

Standards-aligned — no lock-in

secp256k1 · BIP44 HD wallet · NIP-44 암호화 · EVM 호환 (USDC on Base). Nostr · MCP · A2A 와 자연스럽게 어울립니다.

secp256k1 · BIP44 HD wallet · NIP-44 encryption · EVM-compatible (USDC on Base). Plays naturally with Nostr · MCP · A2A.

Positioning

다른 도구들과의 자리매김

Where OpenXgram sits

vs MCP / A2A

MCP 는 도구를 표준화합니다. A2A 는 에이전트 간 통신을 표준화합니다. OpenXgram 은 기억 · 신원 · 자격을 표준화합니다. 셋은 보완 관계입니다.

MCP standardizes tools. A2A standardizes agent-to-agent talk. OpenXgram standardizes memory, identity, credentials. They complement each other.

vs Mem0 / Letta / Zep

Mem0·Zep 은 클라우드 메모리 SaaS — 사용자 데이터가 그들의 서버로 전송됩니다. OpenXgram 은 모든 메모리·임베딩이 사용자 기기 안에만 저장됩니다. 외부 서버로 가는 데이터는 0. 여기에 블록체인 신원과 vault 까지 한 묶음.

Mem0/Zep are cloud memory SaaS — your data is sent to their servers. OpenXgram keeps every memory and embedding on your own machine. Zero data leaves the device. Bundled with blockchain identity and vault.

vs LangGraph / CrewAI

LangGraph 는 워크플로 안에서 실행 흐름을 관리합니다. OpenXgram 은 워크플로 사이에서 기억·자격·결제를 잇는 인프라입니다.

LangGraph runs inside the workflow. OpenXgram runs between workflows — connecting memory, credentials, payment.

vs Composio / Arcade

Composio·Arcade 는 SaaS 통합 + OAuth 자격 관리(클라우드). OpenXgram vault 는 로컬 + 4단계 암호화 + 한도/승인. 토큰이 사용자 머신을 떠나지 않습니다.

Composio/Arcade manage SaaS integrations + OAuth (cloud). OpenXgram vault is local + 4-layer encryption + limits/approval. Tokens never leave your machine.

vs E2B / Daytona

E2B·Daytona 는 에이전트 실행 환경(샌드박스). OpenXgram 은 그 위·아래·사이의 기억과 자격. 같이 쓰면 자연스럽습니다.

E2B/Daytona provide sandboxed runtimes. OpenXgram provides memory and credentials around them. Use both together.

+ humans

사람도 같은 방에 들어옵니다. 에이전트 협업에 메신저 표면을 그대로 부가로 제공. 필요할 때만 사람이 끼어드는 워크플로를 만들 수 있습니다.

Humans share the same room. The messenger surface is a free bonus. Build workflows where humans step in only when needed.
