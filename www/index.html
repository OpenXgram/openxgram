Three core values

Cross-LLM

어떤 AI 도 같은 기억

Same memory across any AI

내 ChatGPT 와 Claude 가 같은 채팅방에 있어 같은 기억을 공유합니다. 모델을 바꿔도 어제 했던 얘기가 그대로 이어집니다.

Your ChatGPT and Claude share the same chat room and the same memory. Switch models and yesterday's conversation just continues.

Cross-device

어디서나 이어가기

Pick up anywhere

휴대폰에서 켜도, 노트북에서 켜도 똑같은 대화·기억·파일. 기기를 바꿔도 처음부터 다시 설명할 필요가 없습니다.

Open it on your phone, your laptop — same chats, same memory, same files. Switch devices without re-explaining yourself.

Safe credentials & payments

비밀번호·API 키는 한 곳에서 관리. 에이전트가 내 카드로 결제할 때 한도와 승인 단계를 정해두면 알아서 안전하게 씁니다.

Passwords and API keys live in one place. Set limits and approval steps, and agents pay safely on your behalf.

Why

왜 OpenXgram 인가?

Why OpenXgram?

  • 모델 락인 없음 — Claude 든 ChatGPT 든 로컬 모델이든 동일한 노트를 봅니다.
  • 기기 락인 없음 — 노트북 → 휴대폰 → 새 컴퓨터, 어디서 켜도 그대로.
  • 로컬 우선 — 기억과 키는 내 기기에 저장. 외부 서비스에 의존하지 않습니다.
  • 변조 검출 — 모든 기록은 영구 봉인. 누가 손대면 즉시 알 수 있습니다.
  • No model lock-in — Claude, ChatGPT, local models all read the same notes.
  • No device lock-in — laptop → phone → new computer, everything stays.
  • Local first — memory and keys stay on your device. No third-party dependency.
  • Tamper-evident — every record is sealed; any change shows up immediately.
