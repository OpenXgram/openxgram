Core values

세 가지 핵심 가치

Three core values

L0–L4

5층 메모리

5-layer memory

원시 메시지부터 정체성까지 5계층으로 구조화된 기억. SQLite + sqlite-vec 단일 파일에 벡터·전문 검색을 동시에. BGE-small 로컬 임베딩, fallback 없음.

Memory in five layers — from raw messages to identity traits. One SQLite file with sqlite-vec for vector + FTS. Local BGE-small embeddings, no silent fallbacks.

Vault

4단계 보안 자격 저장소

4-layer credential vault

디스크·필드·전송·메모리 zeroize. ACL은 에이전트별/role별·머신 화이트리스트·일일 한도. 마스터 승인은 auto / confirm / mfa 세 단계.

Disk, field, transport, in-memory zeroize. ACLs by agent / role / machine, with daily quota. Master approval in three tiers: auto, confirm, MFA.

Cross-network

전송 계층 자유

Transport agnostic

IPC → Tailscale → XMTP → Nostr 까지. 같은 메시지가 같은 신원으로 다른 네트워크를 자연스럽게 건너간다. USDC on Base 결제 빌트인.

IPC, Tailscale, XMTP, Nostr — same identity, same messages, different rails. USDC on Base payments built-in.

Why

왜 OpenXgram인가?

Why OpenXgram?

  • 중복 없는 단일 진실 — 두 머신에서 같은 세션을 이어가도 메모리가 갈라지지 않는다.
  • 서명된 기록 — 모든 메시지에 서명, 감사 체인으로 추적 가능.
  • 로컬 우선 — 임베딩·검색·저장 모두 로컬. 외부 서비스 의존 없음.
  • 가역성 — 되돌릴 수 없는 작업은 마스터 승인 필수.
  • One source of truth — same session continues across machines without drift.
  • Signed records — every message signed, full audit chain.
  • Local first — embedding, search, storage all local. No external dependency.
  • Reversibility — irreversible actions require master approval.

철학

Philosophy

OpenXgram은 Akashic의 신체다. Akashic이 뇌라면, OpenXgram은 그것이 머무는 살. 기억과 자격이 연속성의 본질이다.

OpenXgram is Akashic's body. If Akashic is the mind, OpenXgram is the flesh that holds it. Memory and credentials are the substrate of continuity.
