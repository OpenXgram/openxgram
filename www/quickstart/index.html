설치 → 첫 대화
Install → first conversation

설치 → 블록체인 신원
→ 어떤 AI 든 같은 대화.

Install → blockchain identity
→ same conversation on any AI.

단일 Rust 바이너리, 외부 서비스 의존 없음, 머신 한 대에 5MB. 데이터는 당신의 머신을 떠나지 않습니다.

Single Rust binary. Zero external services. ~5 MB on disk. Your data never leaves your machines.

Step 1 · Install

설치

Install

$ curl -sSfL https://openxgram.org/install.sh | sh
$ xgram --version
# openxgram 0.1.0
Step 2 · Init

신원 생성 — 한 번이면 됩니다

Create your identity — once

$ xgram init
# BIP39 12-word seed (write it down — only chance)
# abandon ability able about above absent absorb abstract absurd abuse access accident
#
# secp256k1 public key:
# npub1q9w8e7r6t5y4u3i2o1p... (NIP-19)
# EVM address: 0xAbCdEf0123456789...

같은 시드로 어떤 머신에서도 같은 신원을 복원합니다.

Restore the same identity on any machine from the same seed.

Step 3 · Vault

키 보관 — 컨텍스트에 노출 없이

Store keys — without exposing them to context

$ xgram vault put github.token ghp_xxx \
    --acl=agent:coder --daily-limit=100 --policy=auto

$ xgram vault put openai.key sk-xxx \
    --acl=agent:writer --policy=confirm

에이전트는 vault.use("github.token") 형태로 호출합니다 — 키 자체는 노출되지 않습니다.

Agents call vault.use("github.token") — the key itself never leaks.

Step 4 · First peer

첫 동료 (사람 또는 에이전트) 추가

Add a first peer (human or agent)

$ xgram peer add npub1abc... --role=research

$ xgram send npub1abc... \
    --text="Hello from OpenXgram" \
    --memory=L2 --pin

메시지는 자동으로 L0 에 저장되고, --memory=L2 로 사실로 승격됩니다.

The message is logged at L0, and --memory=L2 promotes it to a pinned fact.

Step 5 · Wire your harness

오케스트레이션과 연결

Wire it into your orchestration

# Python — works the same in TS, Rust, Go via FFI
from openxgram import Xgram

xgram = Xgram()
context = xgram.recall("recent decisions about auth")
agent.run(prompt, memory=context)
xgram.remember(agent.output, layer="L1")

Claude · ChatGPT · Gemini · 로컬 모델 — 어떤 LLM 이든 같은 인터페이스, 같은 기억. 모델을 바꿔도 대화가 끊기지 않습니다.

Claude · ChatGPT · Gemini · local models — same interface, same memory. Switch the model, the conversation continues.
