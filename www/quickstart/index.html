Quickstart

5분 안에 첫 메시지부터 vault까지.

From first message to vault in five minutes.

모든 명령은 단일 바이너리 oxg로 실행됩니다. 네트워크 호출은 명시적으로만 일어납니다.

Everything runs through one binary, oxg. No surprise network calls.

1 · 설치

1 · Install

현재는 소스 빌드를 권장합니다 (alpha).

Source build for now (alpha).

$ git clone https://github.com/openxgram/openxgram
$ cd openxgram
$ cargo build --release
$ install target/release/oxg ~/.local/bin/

2 · 초기화

2 · Initialize

BIP39 시드를 생성하고 로컬 vault·메모리 DB를 만듭니다.

Generates a BIP39 seed and creates the local vault + memory DB.

$ oxg init
# → ~/.openxgram/seed.enc , memory.db , vault.db 생성
$ oxg whoami
npub1xxxx... (machine: kai-mbp)

3 · 첫 메시지

3 · First message

L0 messages → L1 episode가 자동 생성됩니다.

L0 messages auto-bind into an L1 episode.

$ oxg say "hello, world"
$ oxg recall --query "hello"
[L0 #1] hello, world          (sig ok, episode ep-001)

4 · Vault put / get

4 · Vault put / get

API 키를 안전하게 저장하고, 정책에 따라 다른 에이전트에 노출합니다.

Store an API key, expose it to other agents per policy.

$ oxg vault put openai.api_key
# prompt: paste the secret (echo off)
$ oxg vault get openai.api_key --as agent:eno
sk-proj-xxxxxxxxxxxx
$ oxg vault policy set openai.api_key --tier confirm --daily-limit 100

5 · Peer 추가

5 · Add a peer

다른 머신·에이전트와 같은 기억을 공유하려면 peer로 등록합니다.

Share memory with another machine or agent by adding it as a peer.

$ oxg peer add nostr://npub1abc...xyz --label workstation
$ oxg peer ls
workstation   nostr://npub1abc...xyz   trusted
$ oxg sync push --to workstation
[ok] 124 messages, 12 memories signed and pushed
